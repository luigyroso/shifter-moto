# SHIFTER MOTO — shifter.com.co

Premium homepage mockup and product pages for **Shifter Moto**, a Colombian adventure motorcycle accessories brand.

## Live Site

🔗 **[shifter-moto.github.io](https://luigyroso.github.io/shifter-moto/)**

## Stack

- Pure HTML, CSS, JavaScript — no frameworks, no dependencies
- Google Fonts (Bebas Neue + Inter)
- Product images served from Shifter's Shopify CDN
- Hero background video from Mixkit (free license)

## Pages

- `index.html` — Homepage with video hero, product showcase, product grid, Instagram grid
- `product.html` — Dynamic product page with gallery, variants, and CTA (routed via `?p=slug`)
